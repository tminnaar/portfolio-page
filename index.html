A webpage about music production! It lists some of the equipment I use and a bit of information regarding my workflow. I used flexbox for the first time here and separated all the sections into columns. I've also added in embedded SoundCloud players.
Using today.get I declared values for the day, month and year. Then using DOM Manipulation and expressions for each value I pushed the string into each paragraph element. Media queries were used to then alter the display at descending screen width intervals to reduce the number of cards on the screen and their widths and alter the font sizing of the headings.

A simple comment box with Name and Email Address fields and a comment area. The comment text area has a character limit of 140 that will turn red when exceeded, as well as a required attribute in all fields. On submit, comments are displayed below and persist when a new comment is added.
Using a form element I created inputs for Name, Email Address and a textarea for the comment. Focus and Active states were used to style and highlight the currently active input. Using DOM Manipulation the saved comments are added below using .appendChild. I also utilised alerts to indicate when the character limited is exceeded while trying to submit.

A simple interface using JavaScript objects to display film posters in a slider with information panels on the left. The films can be sorted Alphabetically using a button above the slider and navigated through using directional arrows. Reviews can be added and edited and are stored within the key for each film and persist when navigating and sorting.
Using Object.keys.sort I mutated the movie data object to return an array of keys to establish an index from which to sort and navigate. This enabled me to manipulate content when navigating or sorting. .setAttribute enabled me to push image source links from object property values into the element and alter the background-colour. dataset.mode enabled me to create states for editing or displaying the reviews and window.confirm prevented reviews being saved when navigating or sorting in edit mode.

A single page portfolio with tab navigation and dynamic content. When navigating, panels are displayed on the left containing information for each project. This is my second attempt at a portfolio page, this time around I aimed to make the page more dynamic by repositioning elements and by utilising a variety of animations.
I used keyframe animations to animate the header and buttons sliding in at the bottom, the navigation panel sliding up and quick vertical fade-ins for the panels. For the navigation I added an event listener on each tab that parsed in the relevant panel. It then called a function on it using Array.from and .forEach that set the display of all panels to "none" and the relevent one to "flex".

A reaction time-based game with each rounding lasting 30 seconds. The goal is to click as many targets within the allotted time while the targets randomly resize and reposition. The panel on the right displays the time remaining, current score and high score which are updated using DOM Manipulation. setInterval is used to countdown the time remaining and update the value in the display panel. setTimeout stops the game, removes the target and allows the user to start again.
